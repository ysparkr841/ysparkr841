# 안녕하세요, 박의수입니다 👋

프론트엔드를 중심으로 Java 백엔드까지 아우르는 4년차 개발자입니다.  
데이터 시각화·지도 서비스 구현부터 운영·장애 대응까지 서비스 전체 흐름을 경험했습니다.

<br/>

## 🛠 Tech Stack

**Frontend** &nbsp; Vue.js · JavaScript · React · TypeScript · Chart.js · D3.js · Naver Maps API  
**Backend** &nbsp;&nbsp; Java · REST API · SQL  
**Infra** &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Linux · Jenkins · Tomcat  
**Database** &nbsp;Oracle · MySQL · PostgreSQL  
**Data** &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Talend ETL  
**AI** &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Ollama · LangChain · LangGraph · Qdrant
